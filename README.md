# Zooplankton Hierarchical Classification

## Overview
This repository implements hierarchical classification models for automated zooplankton species identification from microscopic images. Using CNN-RNN architectures, the system can classify zooplankton at multiple taxonomic levels. Hierarchical classification is a complex machine learning task which requires classification at several levels. One aims to generate correct predictions at all hierarchical levels, allowing a singular model to be trained and used for many different applications. To this end we implement a hybrid CNN-RNN model which utilizes the CNN layer to encode images, and treats the hierarchical classification task as next-token prediction through the RNN layer, reusing embeddings generated by the parent-level classification to generate a child-level classification.

## Dataset
Our main dataset is a Zooplankton dataset taken from several great lakes in Ontario. Images have been hand classified by experts according to the following 13 classes `Bosminidae`, `Daphnia`, `Holopedium`, `Rotifer`, `Nauplius_Copepod`,`Cyclopoid`, `Harpacticoid`, `Calanoid`, `Bubbles`, `Exoskeleton`, `Fiber_Hairlike`, `Fiber_Squiggly`, and `Plant_Matter`.
